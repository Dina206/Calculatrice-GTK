# Calculatrice GTK

Calculatrice graphique développée en langage C avec la bibliothèque GTK.

## Fonctionnalités
- Addition
- Soustraction
- Multiplication
- Division
- Interface graphique interactive

## Technologies utilisées
- C
- GTK

## Objectif
Projet académique réalisé dans le cadre de la formation DUT Informatique et Développement Digital.
